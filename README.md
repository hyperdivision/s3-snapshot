# `s3-snapshot`

>

## Usage

```js
var S3Snapshot = require('s3-snapshot')

```

## API

### `const snap = S3Snapshot(directory, s3)`

Create new instance that will snapshot `directory` using a `s3-storage` instance

### `snap.download(key, cb)`

### `snap.snapshot(cb)`

## Install

```sh
npm install s3-snapshot
```
